.Net se compose de 2 types différents : type valeur type référence

Type valeur = type dans lequel on stock une valeur (représente la données qu'il a en mémoire)

Type référence = les données sont stockée en mémoire et un pointeur vers ces données  est stocké dans la variable

String != type valeur

Un type référence doit etre instancie avec le mot NEW

Avec le symbole @ on peut utiliser des mot clés en tant qu'identifiant

Espace de noms : fichier dans lequel on déclare un namespace dans lequel on stock des classes
